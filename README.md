# 🛒 Ekart - Full Stack Ecommerce Website

Ekart is a modern full-stack ecommerce web application built using React.js.

## 🚀 Features

* Responsive UI
* Product Listing
* Search Functionality
* Shopping Cart
* User Authentication
* Category Navigation

## 🛠️ Tech Stack

* React.js
* JavaScript
* CSS
* Material UI
* Node.js (Upcoming)
* Express.js (Upcoming)
* MongoDB (Upcoming)

## 📦 Installation

```bash
git clone https://github.com/saurabhraj18/Ekart.git
cd Ekart
npm install
npm start
```
